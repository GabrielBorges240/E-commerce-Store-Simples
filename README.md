# E-Commerce Simples

Loja virtual com catálogo de produtos e carrinho funcional, simulando um fluxo básico de compra.

Demo:  https://gabrielborges240.github.io/E-Commerce-Simples/

## Funcionalidades
- Listagem de produtos
- Adição e remoção do carrinho
- Cálculo automático do total
- Interface responsiva

## Tecnologias
- HTML5
- CSS3
- JavaScript

## Como executar
Abra o arquivo `index.html` no navegador.
